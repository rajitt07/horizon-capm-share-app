This report estimates beta (β) and alpha (α) for each BSE sector index against the NIFTY Total Return Index using weekly returns. Beta measures the sector’s sensitivity to the market; alpha is the average weekly return in excess of the CAPM-implied level. All calculations use OLS regression; results are aligned by week-end date (Friday) and use only weeks present in both the benchmark and the sector series.

1. Data & methodology

Benchmark (market return): NIFTY Total Return Index. Weekly return = sum of daily "Change %" over the week (Monday–Friday), converted to decimal. Source: Data/industry data/test data/Analytics data/NIFTY Total Returns Historical Data_weekly_change_pct.csv. Columns: Week_End_Date, Weekly_Return.

Sector returns: BSE sector indices. Weekly return = (Close this Friday − Close previous Friday) / Close previous Friday (decimal). Source: Data/sectoral indice beta calculation/3Y_weekly/ (one CSV per sector). Columns: Week_End_Date, Weekly_Return.

Alignment: For each sector, only weeks with a common Week_End_Date in both the benchmark and the sector file are used; no interpolation. Week-end date is Friday.

Model: For each sector we run OLS: Rsector,t = α + β Rmarket,t + εt. Then β = Cov(Rsector, Rmarket) / Var(Rmarket), α = μsector − β μmarket. R², standard errors, t-statistics and p-values (two-tailed, normal approximation) are reported.
