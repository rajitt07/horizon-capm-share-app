Simple Composite Index

Components: Index of Industrial Production, Credit, Broad Money (M3), 10-Year G-Sec (FBIL). Weights below; mean = 100.

How the index is computed
1. Z-score each series: z = (x − mean) / std
2. Raw index = w₁×z_IIP + w₂×z_Credit + w₃×z_M3 + w₄×z_10Y
3. Simple Composite Index = 100 + (Raw index − mean(Raw index))
